# 開発者用仕様書（仮）

## データ構造
このプログラムでは、Apex Legends のキャラ能力を管理するために apexData配列を使用し、各キャラをid、name、passive、senzyutu、ultimate に登録している。


```html
let apexData = [
  { id: 1, 
    name: "キャラ名", 
    class: "クラス名",
    passive: "パッシブ",
    senzyutu: "戦術",
    ultimate: "アルティメット",
  },
]:
```
## ページ遷移

### ページ遷移の仕組み

キャラ一覧、キャラ名、追加などをクリックすると、各ページへ遷移する．
追加，編集，削除から→ 入力フォームページ → 完了ページへ遷移する。

### HTTPメソッドとリソース名
| 操作 | HTTPメソッド | リソース名 | 説明 |
|------|--------------|------------|------|
| 一覧　| get | `/apex` | 全キャラのデータを取得する |
| 詳細表示 | get | `/apex/:id` | 指定IDのキャラ詳細を取得する |
| 新規登録 | post | `/apex` | 新規キャラデータを登録する |
| キャラを編集 | post | `/apex/update/:id` | キャラデータを更新する |
| キャラを削除 | get | `/apex/delete/:id` | 指定キャラデータを削除する |

### 追加・削除・編集後に表示する内容

追加・消去後に確認画面→「完了した」を表示する．
編集後はキャラ一覧を表示する．


## リソースごとの機能の詳細

### get /apex
全キャラクターの一覧データを取得する．
一覧ページの表示に使用し，キャラ名やアビリティの概要を表示する．

### get /apex/:id

指定IDのキャラクター詳細を取得する．
パッシブ・戦術・アルティメットなど、キャラの情報を表示する．

### post /apex

新しいキャラを登録する．
送信されたデータをサーバが受け取り，新規IDを割り当てて保存する．

### post /apex/update/:id

キャラの情報を更新する．
編集から送られた内容をもとに，指定IDのデータを上書きする．

### get /apex/delete/:id

指定したキャラクターのデータを削除する．
削除後は一覧データを再取得し，一覧ページに戻る．
